  - 다중 선택 가능

  - 최소 점수 설정
  - 슬라이더 또는 버튼 선택
  - 80점 이상, 60점 이상 등 프리셋

- **영양소 필터** (고급)
  - 단백질 함량
  - 지방 함량
  - 탄수화물 함량
  - 나트륨 함량

- **메뉴 타입 필터**
  - 순살
  - 뼈 있는 치킨
  - 양념
  - 프라이드
  - 간장
  - 허니 등

#### 2.2 정렬
- **정렬 옵션**
  - 건강 점수 높은 순
  - 건강 점수 낮은 순
  - 칼로리 낮은 순
  - 칼로리 높은 순
  - 단백질 많은 순
  - 가나다순
  - 최신 등록순

#### 2.3 메뉴 카드
각 메뉴 카드 표시 정보:
- 메뉴 이미지 (기본 이미지 제공)
- 메뉴명
- 브랜드명
- 건강 점수 (색상 코드)
  - 80-100: 초록색 (매우 건강)
  - 60-79: 노란색 (양호)
  - 40-59: 주황색 (보통)
  - 0-39: 빨간색 (주의)
- 칼로리
- 단백질
- 나트륨
- 하트 아이콘 (즐겨찾기 - 로컬 스토리지)

#### 2.4 페이지네이션
- 한 페이지당 20개 메뉴
- 무한 스크롤 또는 페이지 번호
- 현재 페이지 표시

## 🔍 메뉴 상세

### 3. 메뉴 상세 페이지

#### 3.1 기본 정보
- **좌측 영역**
  - 메뉴 이미지 (크게 표시)
  - 브랜드 로고

- **우측 영역**
  - 메뉴명
  - 브랜드명 (링크)
  - 건강 점수 (큰 원형 게이지)
  - 점수 설명
  - 즐겨찾기 버튼

#### 3.2 영양 정보

**기본 영양소 (영양성분표 형태)**
```
┌────────────────────────────────────┐
│  영양성분표 (1인분 기준)            │
├────────────────────────────────────┤
│  열량        850 kcal              │
│  탄수화물     45g                  │
│  당류        12g                   │
│  단백질      52g                   │
│  지방        48g                   │
│  ├ 포화지방   12g                  │
│  └ 트랜스지방  0.5g                │
│  콜레스테롤   180mg                │
│  나트륨      1,800mg (90% DV)      │
└────────────────────────────────────┘
```

**상세 영양소 (탭 형태)**

1. **비타민 탭**
   - 비타민 A, B군, C, D, E, K
   - 일일 권장량 대비 %
   - 프로그레스 바

2. **미네랄 탭**
   - 칼슘, 철, 마그네슘, 아연 등
   - 일일 권장량 대비 %
   - 프로그레스 바

3. **필수 아미노산 탭**
   - nutrition-checker 참고
   - 9가지 필수 아미노산
   - 함량 및 % 표시

4. **필수 지방산 탭**
   - 오메가-3, 오메가-6
   - 비율 및 함량

#### 3.3 건강 점수 상세

**점수 구성 (원형 차트)**
```typescript
// 건강 점수 분해도
{
  칼로리점수: 12/15점,
  탄단지균형: 13/15점,
  나트륨점수: 6/10점,
  단백질품질: 8/10점,
  비타민점수: 7/10점,
  미네랄점수: 6/10점,
  식이섬유: 4/10점,
  필수영양소: 8/10점,
  건강한지방: 3/5점,
  저당점수: 4/5점
}
총점: 71/100
```

**건강 점수 해석**
- 점수별 의미 설명
- 이 메뉴의 장점
- 이 메뉴의 단점
- 섭취 시 주의사항

#### 3.4 영양소 시각화

**차트 종류**

1. **탄단지 비율 (파이 차트)**
   - 탄수화물, 단백질, 지방 칼로리 비율
   - 이상적인 비율과 비교

2. **주요 영양소 (레이더 차트)**
   - 단백질, 비타민, 미네랄, 식이섬유 등
   - 일일 권장량 대비 채워진 정도

3. **비타민/미네랄 (바 차트)**
   - 각 영양소별 함량
   - 일일 권장량 대비 %

#### 3.5 비슷한 메뉴 추천
- 같은 브랜드의 다른 메뉴
- 더 건강한 대체 메뉴
- 비슷한 칼로리대의 메뉴

#### 9.4 드래그 스크롤 섹션 🎯

**nutrition-checker의 독특한 UX**

**자동 스크롤 + 드래그 가능**
```
음식 추천 섹션:
┌─────────────────────────────────────────────┐
│ ← [카드1] [카드2] [카드3] [카드4] ... →     │
│   자동으로 2초마다 한 카드씩 스크롤          │
│   드래그로 직접 스크롤 가능                  │
│   마우스 오버 시 자동 스크롤 정지            │
└─────────────────────────────────────────────┘
```

**구현 기능**:
1. **자동 스크롤**
   - 2초마다 자동으로 다음 카드로 이동
   - 끝에 도달하면 처음으로 돌아감
   - `scrollBy({ left: cardWidth, behavior: 'smooth' })`

2. **드래그 스크롤**
   - 마우스 드래그로 좌우 스크롤
   - 터치 디바이스에서도 스와이프 지원
   - 드래그 중 클릭 이벤트 방지

3. **인터랙션**
   - 마우스 오버: 자동 스크롤 정지
   - 마우스 리브: 2초 후 자동 스크롤 재개
   - 스크롤 중: 자동 스크롤 정지, 3초 후 재개

**적용 위치**:
- 인기 메뉴 섹션
- 브랜드 목록
- 비슷한 메뉴 추천
- (Phase 2) 영양제/광고 섹션

**JavaScript 예시**:
```javascript
function initDragScroll(container) {
  let isDown = false;
  let startX;
  let scrollLeft;
  let autoScrollInterval;

  // 자동 스크롤
  const startAutoScroll = () => {
    autoScrollInterval = setInterval(() => {
      const cardWidth = 220; // 카드 너비 + 간격
      if (container.scrollLeft + container.clientWidth >= container.scrollWidth) {
        container.scrollTo({ left: 0, behavior: 'smooth' });
      } else {
        container.scrollBy({ left: cardWidth, behavior: 'smooth' });
      }
    }, 2000);
  };

  // 드래그 스크롤
  container.addEventListener('mousedown', (e) => {
    isDown = true;
    clearInterval(autoScrollInterval);
    startX = e.pageX - container.offsetLeft;
    scrollLeft = container.scrollLeft;
  });

  // ... (나머지 이벤트 핸들러)

  startAutoScroll();
}
```

**화면 크기별 광고 전략**

**초대형 (1600px+)**
```
[좌측 광고 300px] [메인 1200px] [우측 광고 300px]
```

**대형 (1200px ~ 1599px)**
```
[메인 컨텐츠 1400px, 광고 없음]
```

**중형 (768px ~ 1199px)**
```
[메인 컨텐츠 100%, 광고 없음]
```

**모바일 (< 768px)**
```
[메인 컨텐츠]
[하단 고정 광고 100% × 100px]
```

**모바일 하단 광고**:
- `position: sticky; bottom: 0`
- 스크롤해도 항상 보임
- 닫기 버튼 (선택)
- 320px × 100px (모바일 배너 표준)

### 12. 영양소 프로그레스 바 📊

**nutrition-checker 스타일**

**4단계 색상 코드** (일일 권장량 대비)
```
80% 이상:  ████████ 초록색 (#48bb78) - 충족
50-79%:    ██████░░ 연두색 (#9ae6b4) - 충분
20-49%:    ███░░░░░ 주황색 (#f6ad55) - 불충분
20% 미만:  █░░░░░░░ 빨간색 (#fc8181) - 부족
```

**프로그레스 바 컴포넌트**:
```html
<div class="nutrient-item">
  <div class="nutrient-header">
    <span class="nutrient-name">이소류신</span>
    <span class="nutrient-status status-excellent">충족</span>
  </div>
  <div class="nutrient-bar">
    <div class="nutrient-progress" style="width: 85%; background: #48bb78;"></div>
  </div>
  <div class="nutrient-footer">
    <span class="nutrient-value">1190 mg</span>
    <span class="nutrient-target">권장: 1400 mg</span>
  </div>
</div>
```

**탭별 부족 영양소 인디케이터**
- 탭 버튼에 빨간 숫자 배지
- 50% 미만 영양소 개수 표시
- 예: `필수 아미노산 (3)` ← 3개 부족
