# 치킨은 살 안 쪄 🍗

> 전국 치킨 메뉴 영양 분석 플랫폼

## 📋 프로젝트 개요

**치킨은 살 안 쪄**는 대한민국 치킨 프랜차이즈 메뉴를 영양학적으로 분석하여, 사용자가 치킨을 선택할 때 영양 정보와 건강 점수를 확인할 수 있는 웹 애플리케이션입니다.

### 핵심 가치

- **공신력 있는 데이터**: 브랜드 공식 데이터, 식품안전나라, 한국소비자원 등 검증된 출처만 사용
- **투명한 영양 정보**: 치킨 메뉴별 38종 영양소 상세 분석
- **건강 점수 시스템**: 자체 개발한 건강 점수 알고리즘으로 메뉴 비교 (100점 만점)
- **시각화**: 3대 영양소 파이 차트, 영양소별 상태 표시
- **데이터 품질 등급**: A/B/C 등급으로 데이터 신뢰도 표시

### 현재 상태 (2025-10-16)

- ✅ **프론트엔드 구축 완료**: React 18 + TypeScript + Vite
- ✅ **디자인 시스템**: 세련된 치킨 테마, 다크/라이트 모드
- ✅ **12개 메뉴 등록**: BHC(2), 교촌(3), 푸라닭(3), 호식이(3), 60계(1)
- ✅ **100% 검증된 메뉴**: 모든 메뉴 실존 확인 완료
- ✅ **건강 점수 알고리즘**: 감점/가점 요소 기반 100점 만점 시스템
- ✅ **메뉴 이미지**: 각 브랜드 공식 이미지 연동

## 🎯 주요 기능

### 1. 치킨 메뉴 데이터베이스
- **12개 메뉴 등록** (2025-10-16 기준)
  - **BHC (2개)**: 뿌링클, 맛초킹
  - **교촌 (3개)**: 레드+허니갈릭, 레드 한마리, 허니 한마리
  - **푸라닭 (3개)**: 씬 후라이드, 달콤양념 치킨, 투움바 치킨
  - **호식이 (3개)**: 후라이드, 양념치킨, 간장치킨 (시그니처)
  - **60계 (1개)**: 간지치킨 순살
- 브랜드별/메뉴별 검색 및 필터링
- 브랜드 공식 제품 이미지 표시

### 2. 영양 분석 (38종 영양소)
- **3대 영양소**: 칼로리, 탄수화물, 단백질, 지방
- **지방 상세**: 포화지방, 트랜스지방, 불포화지방
- **필수 아미노산 9종**: 이소류신, 류신, 발린, 라이신, 메티오닌, 페닐알라닌, 트레오닌, 트립토판, 히스티딘
- **필수 지방산**: 리놀레산, 알파리놀렌산, EPA, DHA
- **비타민 9종**: A, D, E, B1, B2, B3, B6, B12, C
- **미네랄 7종**: 칼슘, 철분, 아연, 마그네슘, 인, 칼륨, 셀레늄
- **기타**: 식이섬유, 당류, 콜레스테롤, 나트륨

### 3. 건강 점수 시스템
- **100점 만점** 건강 점수 알고리즘
- **감점 요소** (최대 -60점): 포화지방, 트랜스지방, 콜레스테롤, 나트륨, 설탕, 탄수화물
- **가점 요소** (최대 +30점): 단백질, 식이섬유, 비타민, 미네랄
- **5단계 색상 표시**:
  - 80점 이상: 짙은 초록 (우수)
  - 70-79점: 부드러운 초록 (좋음)
  - 60-69점: 살구색 (보통)
  - 50-59점: 테라코타 (주의)
  - 50점 미만: 차분한 빨강 (나쁨)

### 4. 시각화
- **파이 차트**: 3대 영양소(단백질/탄수화물/지방) 칼로리 비율
- **영양소 상태바**: 각 영양소별 권장량 대비 섭취량 표시
- **데이터 품질 배지**: A/B/C 등급 표시
- **다크/라이트 모드**: 토글 스위치로 테마 변경
- **Glass-morphism**: 현대적인 유리 효과 디자인

## 🛠 기술 스택

### Frontend (구축 완료)
- **React 18**: UI 라이브러리
- **TypeScript 5**: 타입 안정성
- **Vite**: 빌드 도구 (개발 서버 구동 중)
- **Tailwind CSS**: 유틸리티 기반 스타일링
- **SVG**: 파이 차트 직접 구현 (외부 라이브러리 없음)
- **CSS Variables**: 다크/라이트 모드 테마 시스템

### 데이터 관리
- **Mock Data**: TypeScript 파일 기반 데이터 관리 (`mockData.ts`)
- **공식 출처**:
  - 브랜드 공식 웹사이트 (BHC, 교촌, 푸라닭, 호식이, 60계)
  - 식품안전나라 (식품의약품안전처)
  - 한국소비자원 공식 보고서
  - USDA FoodData Central

### Backend (추후 개발 예정)

## 🚀 시작하기

### 필수 요구사항

- Node.js 18 이상
- npm 또는 yarn

### 개발 서버 실행

```bash
# 1. 프로젝트 디렉토리로 이동
cd "C:\Claude\치킨은 살 안 쪄\frontend"

# 2. 의존성 설치 (최초 1회)
npm install

# 3. 개발 서버 실행
npm run dev

# 4. 브라우저에서 접속
# http://localhost:5173
```

### 빌드

```bash
# 프로덕션 빌드
npm run build

# 빌드 결과 미리보기
npm run preview
```
